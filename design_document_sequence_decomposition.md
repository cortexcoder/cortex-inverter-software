# 正负序分解模块设计文档

## 1. 概述

正负序分解模块（Sequence Decomposition Module）用于将三相电压/电流分解为正序、负序和零序分量。这是实现**高低穿算法**（High/Low Voltage Ride-Through）的核心模块。

### 1.1 应用场景

- **电网故障穿越**：当电网发生不对称故障（如单相接地、两相短路）时，产生负序分量
- **不平衡控制**：抑制负序电流，提高电能质量
- **锁相环辅助**：在电网不平衡条件下辅助 PLL 提高锁相精度

### 1.2 模块特性

- 支持三相输入和 αβ 坐标系输入
- 正序和负序 dq 分解
- 可选的零序分量输出
- 与 PLL 无缝集成

## 2. 算法原理

### 2.1 对称分量法 (Fortescue Transformation)

对于任意三相不平衡量 $V_a, V_b, V_c$，可以分解为：

```
V_pos = (V_a + a*V_b + a^2*V_c) / 3
V_neg = (V_a + a^2*V_b + a*V_c) / 3
V_zero = (V_a + V_b + V_c) / 3
```

其中旋转因子：
- $a = e^{j120°} = -0.5 + j\frac{\sqrt{3}}{2}$
- $a^2 = e^{j240°} = -0.5 - j\frac{\sqrt{3}}{2}$

### 2.2 αβ 坐标系下的实现

在 αβ 坐标系中，对称分量可以表示为：

```
V_pos = V_α + j*V_β (在正序旋转坐标系中)
V_neg = V_α - j*V_β (在负序旋转坐标系中等价)
```

更精确地，正负序 dq 分量可以通过 Park 变换得到：

**正序 dq 变换**（使用 PLL 角度 θ）：
```
V_pos_d = V_α * cos(θ) + V_β * sin(θ)
V_pos_q = -V_α * sin(θ) + V_β * cos(θ)
```

**负序 dq 变换**（使用负序角度 -θ 或 θ+π）：
```
V_neg_d = V_α * cos(-θ) + V_β * sin(-θ)
V_neg_q = -V_α * sin(-θ) + V_β * cos(-θ)
```

## 3. 与 PLL 的配合方案

### 3.1 数据流

```
                    ┌─────────────┐
    V_a, V_b, V_c ──>│   Clarke    │──> V_α, V_β
                    └─────────────┘
                           │
                           v
    ┌─────────┐      ┌─────────────────┐
    │   PLL   │─────>│  SeqDecomp      │
    │ (θ_pll) │      │  (正负序分解)   │
    └─────────┘      └─────────────────┘
                           │
              ┌────────────┼────────────┐
              v            v            v
          V_pos_dq     V_neg_dq      V_zero
```

### 3.2 角度同步策略

```
正序角度: θ_pos = θ_pll (来自 PLL 输出)

负序角度: θ_neg = θ_pll + π (旋转方向相反)

或者等价地: θ_neg = -θ_pll (在复平面内共轭)
```

### 3.3 不同运行模式

| 模式 | PLL 状态 | 正序角度来源 | 负序角度处理 |
|------|----------|-------------|-------------|
| 正常锁相 | 锁定 | θ_pll | θ_neg = θ_pll + π |
| 开环延展 | 开环 | 保持的 last_valid_theta | θ_neg = last_valid_theta + π |
| 频率扰动 | 跟踪 | θ_pll (可能偏移) | θ_neg = θ_pll + π |

### 3.4 高低穿场景下的特殊处理

在电网电压跌落/抬升期间：

1. **电压跌落 (Low Voltage Ride-Through)**
   - 正序分量快速减小
   - 负序分量增大（体现电网不平衡）
   - PLL 可能失锁，进入开环延展模式
   - **处理策略**：使用开环角度维持正负序分解，直到 PLL 重新锁定

2. **电压抬升 (High Voltage Ride-Through)**
   - 正序分量快速增大
   - 负序分量可能增大
   - **处理策略**：限幅保护，防止过调制

## 4. 模块接口

### 4.1 初始化

```c
SeqDecomp_Config cfg = {
    .Ts = 0.00005f,      // 50μs (20kHz)
    .fc_hp = 2.0f,       // 高通滤波器截止频率 Hz
    .fc_lp = 2.0f        // 低通滤波器截止频率 Hz
};
SeqDecomp_Init(&h, &cfg);
```

### 4.2 使用流程

```c
// 1. 获取 PLL 角度
float theta_pll, freq;
bool pll_locked = Pll_Step(&pll, v_alpha, v_beta, &theta_pll, &freq);

// 2. 根据 PLL 状态设置正负序角度
if (!pll_locked) {
    // 开环模式：使用保持的角度
    SeqDecomp_SetThetaPos(&seq, open_loop_theta);
} else {
    // 正常模式：使用 PLL 输出
    // SeqDecomp 会自动处理
}

// 3. 执行正负序分解
float pos_d, pos_q, neg_d, neg_q, zero;
SeqDecomp_Step_AlphaBeta(&seq, v_alpha, v_beta, theta_pll,
                         &pos_d, &pos_q, &neg_d, &neg_q, &zero);

// 4. 使用正负序分量进行控制
// - 正序用于正常电流控制
// - 负序用于不平衡抑制
// - 零序用于接地故障检测
```

## 5. 高低穿算法集成建议

### 5.1 LVRT (Low Voltage Ride-Through)

```
电压跌落检测:
│V_pos│ < V_threshold (如 0.8 pu)

控制策略:
1. 正序电流参考: 根据电网规范确定（如 1.1 pu 持续有限时间）
2. 负序电流参考: 通常注入一定比例负序电流以维持电网稳定性
3. 功率控制: 有功 P = f(电压), 无功 Q = k * (1 - 电压)

关键参数:
- 电压跌落阈值: 0.8 pu (可配置)
- 最小保持时间: 150ms (根据电网规范)
- 正序电流限值: 1.2 pu
- 负序电流限值: 0.1 pu (或按规范)
```

### 5.2 HVRT (High Voltage Ride-Through)

```
电压抬升检测:
│V_pos│ > V_threshold (如 1.1 pu)

控制策略:
1. 正序电流参考: 限制在允许范围
2. 负序电流参考: 减小（不平衡程度降低）
3. 功率控制: 有功 P 减小，无功吸收增加

关键参数:
- 电压抬升阈值: 1.1 pu (可配置)
- 最大允许电压: 1.3 pu (硬件限制)
- 正序电流限值: 0.5 pu (限制功率)
```

## 7. 注意事项

1. **零序分量**：在三相三线制系统中，零序电流通常为零（无零序电流通路）。但零序电压检测可用于接地故障诊断。

2. **负序提取精度**：在频率偏移情况下，简单的角度旋转法可能产生误差。建议使用延迟信号法（DSF）提高精度。

3. **计算延迟**：正负序分解应在 ADC 采样后尽快完成，避免引入相位误差。

4. **数值稳定性**：使用 fixed-point 算法时注意防止溢出。
